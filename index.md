# Adapt Apps Builder Console Documentation

Welcome to the documentation for the **Adapt Apps Builder Console**. This guide provides everything you need to know about building powerful forms and data views within the Adapt Apps platform.

## What is the Builder Console?

The Adapt Apps Builder Console is a powerful interface designed to help you create, manage, and deploy applications with ease. Whether you're building simple data collection forms or complex data-driven views, the Builder Console provides the tools you need to succeed.

## Key Features

### [Building Forms](./docs/forms/building_a_form)
Create intuitive and dynamic forms to capture data efficiently. Our Form Builder allows you to add various field types, configure validation, and customise layouts with ease.

### [Building Data Views](./docs/data_views/building_a_data_view)
Transform your data into actionable insights with custom Data Views. Configure columns, set up complex filtering, and define sorting and limiting rules to present your data exactly how you need it.
